== 前言

本书定义 Rootward 项目型 CLI 模板体系的公共契约。该契约适用于需要在文件系统项目中运行、读取项目内配置、发现源文件集合、执行业务扫描并暴露稳定命令输出的开发者工具。

Rootward 的对象不是命令行参数解析样板，也不是业务扫描器。Rootward 固定的是项目型 CLI 反复需要的共同结构：工具身份、隐藏项目目录、初始化命令、项目根发现、TOML 配置、source 声明、gitignore-aware 文件发现、scanner registry、discover 投影、doctor 诊断、稳定 JSON 输出、错误码、退出码和可测试核心。

本书面向三类读者。理解型读者可以通过第一部和第二部理解 Rootward 的对象边界。实现型读者可以通过第三部、第四部和第五部实现语言模板。维护型读者可以通过第六部和附录判断新增设计点是否有资格进入 Rootward 契约。

README 负责提供入口说明。本书负责提供契约说明。入口说明可以压缩，契约说明必须保留对象定义、公共规则、边界条件、实现责任和验证方式。
